DriveSafe Vision

Sistema computacional embarcado para detecção, em tempo real, do uso de celular ao volante, com foco em eficiência, privacidade e relevância social para a segurança viária.

Descrição do Projeto

A distração ao volante, especialmente pelo uso de telefone celular, é um dos principais fatores associados ao aumento de acidentes de trânsito. Nesse contexto, o DriveSafe Vision é um sistema computacional embarcado desenvolvido para detectar, em tempo real, o uso de celular durante a condução veicular, inserido no contexto dos Sistemas Avançados de Assistência ao Motorista (ADAS).

O sistema combina técnicas de visão computacional e deep learning, empregando o modelo YOLO para a detecção visual do telefone celular, aliado à análise temporal para validação dos eventos e redução de falsos positivos. A implementação é realizada em C++ com OpenCV, priorizando eficiência e viabilidade em plataformas embarcadas de baixo custo.

Ao identificar comportamentos de risco, o sistema emite alertas locais e envia notificações remotas via Telegram, respeitando princípios de privacidade e minimização de dados. Como resultado, o projeto busca contribuir para a mitigação de comportamentos de risco e para o aumento da segurança viária.

Destaques

Pontos-chave do DriveSafe Vision, alinhados ao foco embarcado e à robustez do alerta.

Problema: distração ao volante por celular, aumentando riscos de acidentes.
Solução: detecção em tempo real em plataforma embarcada com arquitetura leve.
Pipeline: captura → pré-processamento → segmentação de movimento → morfologia → contornos → análise temporal.
Implementação: C++ + OpenCV, otimizada para baixo custo computacional.
Saída: alertas locais + notificações remotas via API do Telegram, com evidência pontual (sem vídeo contínuo).
Evolução: potencial multimodal com deep learning, VLMs e LLMs no contexto de ADAS.

Testes Experimentais com Imagens

Para a validação do sistema de detecção de uso de telefone celular ao volante, foram realizados testes experimentais utilizando imagens estáticas capturadas no interior de veículos, contendo motoristas em diferentes condições de uso do celular nas Figuras 2 e 3, usando a base de dados Kaggle.

As imagens de teste apresentam cenários realistas, incluindo variações de iluminação, ângulo de captura, posição do motorista e forma de interação com o telefone celular (por exemplo, uso próximo ao rosto durante chamadas).

O algoritmo foi configurado para detectar exclusivamente o objeto telefone celular, ignorando propositalmente outras classes presentes na cena, como rosto, mãos, volante ou partes do veículo. A detecção correta é indicada por uma caixa delimitadora (bounding box) na cor verde, acompanhada do rótulo “CELULAR”.

Conforme observado nas imagens de teste, o sistema foi capaz de localizar corretamente o telefone celular mesmo quando este se encontrava parcialmente ocluído ou próximo ao rosto do motorista, demonstrando robustez frente a contextos comuns de condução.

Esses resultados indicam que a abordagem adotada é adequada para a identificação visual do uso de celular ao volante em ambientes controlados, constituindo uma etapa fundamental para aplicações futuras em sistemas embarcados de monitoramento de segurança veicular.
